## \# 适合的读者

在 Helm 图表的生命周期内执行的钩子数量是没有限制的，并且在某些情况下，可能会为同一生命周期的某个阶段配置多个钩子。此时，钩子的默认执行顺序是按照钩子的首字母顺序。通过 helm.sh/weight 注解可以为每个钩子设置权重，这样就可以按权重顺序执行钩子。权重按升序排序，如果多个钩子的权重值相同，则继续按名称字母顺序执行。
